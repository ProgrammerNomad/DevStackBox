## 🚀 **How to Use (Zero Dependencies):**

### **Step 1:** Start DevStackBox
```bash
npm start
```

### **Step 2:** Download Servers (First Time Only)
- Click the yellow "Download Portable Servers" banner
- Or use menu: Install → Download Portable Servers
- Wait for ~150MB download to complete
- Servers are automatically configured

### **Step 3:** Use DevStackBox
- Start/Stop Apache and MySQL from GUI
- All services use portable binaries
- No XAMPP or other software needed!
