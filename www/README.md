# DevStackBox - Web Root

This is the default document root for your web projects.

## Structure

```text
www/
├── index.php      # Default landing page
├── phpinfo.php    # PHP information page
└── projects/      # Your web projects go here
```

## Usage

- Place your PHP projects in the `projects/` directory
- Access via `http://localhost/projects/yourproject/`
- Default page shows status and links

## Features

- Auto-generated project listings
- PHP info and version display
- Database connection testing
- Development tools and utilities
