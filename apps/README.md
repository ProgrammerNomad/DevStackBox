# DevStackBox - App Installers

This directory contains one-click installers for popular PHP applications.

## Available Installers

- **WordPress** - Popular CMS for websites and blogs
- **Laravel** - Modern PHP framework for web applications
- **CodeIgniter** - Lightweight PHP framework
- **Drupal** - Enterprise CMS (planned)
- **Joomla** - Content management system (planned)

## Features

- One-click installation from GUI
- Automatic database setup
- Pre-configured for local development
- Easy project management

## Structure

```text
apps/
├── wordpress/     # WordPress installer and templates
├── laravel/       # Laravel installer and templates
├── codeigniter/   # CodeIgniter installer
└── templates/     # Common templates and configs
```
